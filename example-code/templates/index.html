Overview

Here is what you can do now:

  • The main application code is in hello.py. This sets up the Flask application and provides a single view that renders this page.
  • The template that is rendered is templates/index.html and contains the source for the website you are just looking at.
  • The layout template that is used is templates/layout.html. It contains the doctype and general HTML structure.
  • The stylesheet and other static files are located in the static folder.
